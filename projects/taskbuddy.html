Task Buddy

Project Overview

Github

Task Buddy is a user-friendly web application that allows users to track their to-do list. Our application allows users to create personal accounts and access their to-do lists stored within the databases.

Programming Languages: HTML, CSS, Javascript, PHP, SQL

Software/Tools: Figma, SQLite

This project is completed by three undergraduate computer science students for our web development course.

Our Goal: Design and develop a todo application that makes tracking todo as friendly as possible to increase the productivity of users.

My Role

My main contributions were designing the user interface, working on the backend database, and connecting everything together.

I worked on:

  • UI Design
  • Creating and updating database using SQLite and PHP
  • Implementing Javascript functionalities such as adding todos
  • Fetching from database and displaying as HTML using PHP and Javascript
  • Connecting front-end and back-end
