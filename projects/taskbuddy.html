Overview

View Code

Task Buddy is a user-friendly web application that allows users to track their to-do list. Our application allows users to create personal accounts and access their to-do lists stored within the databases.

Our Goal: Design and develop a todo application for visual thinkers using Kanban methods.

Toolkit

Programming Languages: HTML, CSS, Javascript, PHP, SQL

Software/Tools: Figma, SQLite

Feature Walkthrough

Login and Signup to the App to Save Todos

Users can log in or sign up to the app to save and manage their to-do lists securely.

Start Page with Greetings and Mood Tracking

The Start Page welcomes users with a personalized message based on the time of day and features a mood tracking tool that sets their profile photo for the day.

Adding To-Dos

Users can effortlessly add new tasks to their to-do list with an intuitive interface.

Kanban Board Style To-Do List

Organize tasks with a Kanban board view, allowing users to manage and visualize their to-do lists. Easily move tasks between different statuses to track progress and prioritize efficiently.
