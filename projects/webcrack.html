Overview

View Code View Devpost

WebCrack is a Chrome extension developed to detect potential bias in online articles by analyzing the background of the source and author. It also offers features like estimating read time and suggesting additional relevant articles to enhance the reading experience.

This project was created for the 2024 Hack for Humanity with the goal of empowering ESL (English as a Second Language) individuals. By providing a tool that simplifies the process of consuming information online, we aim to help users recognize and navigate potential biases and misinformation more effectively in their daily lives.

Toolkit

Programming Languages: HTML, CSS, Javascript, React.js

API: ChatGPT API

Framework: LangChain
