Overview

View Code

SplitZ is a web and mobile app designed to simplify bill splitting among friends. Users can enter or upload a receipt, choose items, and specify how to divide each among multiple people. The app provides flexibility to adjust the number of participants, modify items, and customize individual contributions!

Our Goal: Make splitting bills with friends fast and stress-free!

Toolkit

Programming Languages: HTML, CSS, Javascript

Software: Figma
