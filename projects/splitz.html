Full Stack Mobile App

SplitZ

Project Overview

Github

SplitZ is a web mobile app that makes spliting bills with friends easy! Enter receipt or upload image to start and split by items. User have control over number of people to split bills with, add/edit items, and split each item with mutiple people!

Programming Languages: HTML, CSS, Javascript

Software/Tools: Figma

Our Goal: Make spliting bill with friends fast and stress free!

My Role

My main contributions were designing the user interface, coding the front-end, and connecting everything together.

I worked on:

  • Designing user interface using Figma
  • Developing front-end from Figma to code using HTML, CSS, and JS.
  • Connecting back-end reciept scanner to front-end.
  • Using Javascript to dynamically update page content.
  • Writing Documentations
